# Day 14

My approach here was to simply create a grid to represent the environment, draw the "rock" paths on
it, and then simulate the sand. The grid struct I created for day 12 lends itself well to this, so
that worked out nicely.

The downside of this approach is that for part 2, I had to actually add a floor, and just kind of
guess how big it needed to be (since I can't do infinite, obviously).

An alternative approach would be to have a hash map of occupied points. That way you wouldn't need
to set an explicit width for the floor, but could instead just check manually against the floor's
y coordinate. The width wouldn't matter, since the hash map can contain any coordinate.

But whatever; I got to reuse my Grid, and it got the right answer, and drew some pretty pictures.
First is the grid without sand, then the part 1 grid filled with sand, and the part 2 grid filled
with sand.
